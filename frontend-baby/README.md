# frontend

Aplicación **React** (SPA) para la interfaz de usuario.

## Descripción
Pantallas de autenticación, dashboard, cuidados, gastos, hitos, diario, citas y rutinas. Rutas protegidas con JWT y consumo de APIs vía Axios.

## Tecnologías usadas
- **React**
- **Material UI**
- **React Router DOM**
- **Axios** (interceptor con token JWT)
- **jwt-decode**
- Gestión de estado con Context / useReducer
- Herramientas: Create React App (o Vite), ESLint/Prettier (opcional)
